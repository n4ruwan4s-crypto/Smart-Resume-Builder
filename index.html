⚙️ n8n Webhook Setup Guide
One-time setup · Takes ~15 mins · Auto-detects FriMi payments
1
Create a free n8n cloud account
Go to app.n8n.cloud → Sign up free (500 executions/month). No credit card needed.
2
Create a new Workflow → Add Webhook node
Click + New Workflow → Add node → Search Webhook → Select it.
Set HTTP Method to POST and copy your webhook URL:
https://your-account.app.n8n.cloud/webhook/procv-frimi
3
Set up FriMi email forwarding → Parse payment
FriMi sends you an email for every received payment. Forward those emails to n8n using Gmail trigger or email-to-webhook service (e.g. Zapier Email Parser free tier).

Add a Code node after Webhook. Paste this script to extract amount:
// In n8n Code node: const body = $input.first().json; const amount = body.amount || body.value || '500'; const ref = body.reference || body.txn_id || Date.now().toString(); return [{ json: { amount, reference: ref, paid: true } }];
4
Add HTTP Request node → Notify ProCV
Add HTTP Request node. Set it to POST your payment confirmation to a free store like Pipedream KV store or a simple JSONBin.io bin.

Response payload your webhook sends to ProCV:
{ "paid": true, "amount": "500", "currency": "LKR", "reference": "FRI-XXXXXXXX", "timestamp": "2025-01-01T12:00:00Z" }
5
Paste your webhook URL into ProCV
Once your n8n workflow is live, paste the webhook URL below. ProCV will poll it every 4 seconds after the user scans the QR. When payment is detected → download starts automatically.
✅ Webhook URL saved!
6
Replace placeholder QR with your real FriMi QR
In your FriMi app → Receive Money → Copy QR code image. Host it on GitHub (same repo) or Imgur.
Then in index.html find this line and replace with your image URL:
// Line ~1260 in index.html: // REPLACE src with your actual FriMi QR image URL <img id="frimi-qr-img" src="YOUR_QR_URL_HERE">

🔒 Privacy Policy
✅ We do NOT store your personal data
All CV information you enter stays in your browser only. Nothing is sent to our servers, saved in a database, or shared with any third party.
What happens to my data?
• All CV data is processed entirely in your browser using JavaScript
• Your name, email, phone, work history never leave your device
• Closing the browser tab permanently clears all data
• Generated CV files (PDF / Word) are downloaded directly to your device
• We do not use cookies to track personal information
